# EKG Analyse App - Abschlussabgabe 🫀

Diese Streamlit-Anwendung dient zur Visualisierung, Verwaltung und Analyse von EKG-Daten. Sie wurde objektorientiert und modular in Python entwickelt.

## 🚀 Features
* **Personenverwaltung:** Anzeige von Metadaten (Alter, Max-HF, Profilbild) sowie Hinzufügen und Bearbeiten von Patienten.
* **EKG-Visualisierung:** Interaktive Plotly-Diagramme mit dynamischer Anpassung des Zeitfensters.
* **Optimierte Datenverarbeitung:** Reduktion der Datenauflösung für schnelle Ladezeiten.
* **Signalanalyse:** Berechnung der durchschnittlichen Herzfrequenz und automatisierte Erkennung von Anomalien mittels `neurokit2`.
* **Admin-Dashboard:** Statistische und anonymisierte Auswertung von Patientengruppen.
* **Theme-Switcher:** Integrierter Dark Mode / Light Mode für optimale UI/UX.
